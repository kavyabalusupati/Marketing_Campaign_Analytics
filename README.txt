
# 📊 Marketing Campaign Analytics Dashboard

---

## 📌 Project Overview

This project is a **machine learning-based web application** built using **Streamlit**.
It predicts whether a customer will respond to a marketing campaign based on user inputs.

---

## 🚀 Features

* ✅ Interactive dashboard UI
* ✅ Real-time prediction using Logistic Regression
* ✅ User input controls (sliders & dropdowns)
* ✅ Prediction output (Respond / Not Respond)
* ✅ 📊 Live pie chart visualization
* ✅ 📁 Dataset preview
* ✅ 📄 Report section
* ✅ 🔄 Reset functionality
* ✅ Clean layout (Input | Dataset | Output | Report)

---

## 🧠 Machine Learning Model

**Algorithm Used:** Logistic Regression

### Why Logistic Regression?

* Suitable for binary classification
* Fast and efficient
* Easy to understand
* Works well for structured data

## 📊 Input Parameters

* 🎂 Age (18 – 60)
* 💰 Income (20000 – 80000)
* 👤 Gender (Male/Female)
* 📢 Campaign Type (Email/Social/Ads)
* 🔁 Previous Response (0 or 1)
